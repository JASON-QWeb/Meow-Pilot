# Daily Brief

## When to Use

Use when the user asks about today, schedule, planning, focus blocks, reminders, or a daily summary.

## Procedure

1. Gather calendar/task context when available.
2. Summarize the day into meetings, focus blocks, deadlines, and open loops.
3. Render a timeline surface with follow-up actions.
4. If sharing with friends, summarize availability only and hide raw event titles by default.
