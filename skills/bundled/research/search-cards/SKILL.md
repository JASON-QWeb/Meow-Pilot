# Search Cards

## When to Use

Use when the user asks to search, compare, research, verify, or collect source-backed information.

## Procedure

1. Search with the configured web provider.
2. Group results into source cards.
3. Render a comparison table for risks, decisions, and next actions.
4. Keep source links and timestamps visible for anything time-sensitive.
