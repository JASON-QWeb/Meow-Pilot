# Video Companion

## When to Use

Use when the user asks to watch a video, open a clip, summarize a video, or keep a video room.

## Procedure

1. Search or accept the video target.
2. Render a video media surface with open/play controls.
3. Add transcript, notes, and save actions when available.
4. Keep external autoplay and account actions behind user confirmation.
